# hadist-app With Ai
Tugas UAS Smt 4 STMIK AB

Step menjalakan Project:
Backend: 
1. cd backend
2. npm i
3. npm run start

Frontend:
1. cd frontend
2. npm i
3. npm run dev

Jalankan Backend dan frontend bersamaan.


Depedency:
Backend:

  "express": "^4.18.1",
  "mysql2": "^2.3.3",
  "nodemon": "^2.0.16"

Frontend:
  "next": "^12.1.6",
  "react": "^18.2.0",
  "react-dom": "^18.2.0"

Database: Mysql | clearDB Mysql
Server: Heroku | localhost
